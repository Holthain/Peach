## MongoDB Installation

download mongodb 32 bits from
'https://www.mongodb.org/dl/win32/i386'

Open cmd as administrator and run

```
"\path\to\mongod.exe" --config "\path\to\mongod.cfg" --install

net start MongoDB
```
